# MOVEON LOCADORA – Sistema Web de Locação de Veículos

![banner](https://github.com/gvnff/Projeto-Sistema-Locadora-MoveOn/blob/main/banner.jpg)

A MOVEON LOCADORA é um sistema web desenvolvido para facilitar e automatizar o processo de locação de veículos. O projeto foi construído como parte da disciplina **Projeto de Extensão em Programação**, integrando HTML, CSS, JavaScript e ASP.NET para oferecer uma plataforma acessível, organizada e funcional.

---

##  Funcionalidades Principais

- **Cadastro de Usuários**  
  Permite registrar clientes, funcionários e administradores no sistema.

- **Login e Autenticação**  
  Controle seguro de acesso conforme o perfil do usuário.

- **Busca de Veículos**  
  Pesquisa por categoria, preço, modelo e disponibilidade.

- **Gestão de Aluguéis**  
  Efetua reservas, devoluções e atualiza o status dos veículos.

- **Geração de Contrato**  
  Emite contratos automaticamente após a reserva.

- **Painel Administrativo**  
  Controle de funcionários, veículos e todo o fluxo de locações.

---

##  Estrutura do Sistema

### **1. Cadastro de Veículos**
Armazena dados como modelo, categoria, ano, placa, quilometragem e status de disponibilidade.

### **2. Cadastro de Clientes**
Registra informações essenciais para locação, como nome, documento e contato.

### **3. Gestão de Aluguéis**
Controla reservas, datas, pagamentos e devoluções.

### **4. Administração**
Gerencia funcionários, carros e toda a parte operacional.
